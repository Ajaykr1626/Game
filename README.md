Game Name: Tiki Topple

How to Run:
- Install Python
- Run: python game.py

Rules:
- Players take turns
- Move top 1–3 tokens OR reorder top 2–3 tokens
- Tokens move forward
- Game ends after 20 turns or reaching end

Tech Used:
- Python

Features:
- Turn-based gameplay
- Stack system
- Scoring system
